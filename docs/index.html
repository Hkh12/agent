About agent.js

Agent is a javascript tool for get client's browser and OS information.

Get Started

First, Download it from GitHub. then, Include it into your HTML. 
                    <script src="agent.min.js"></script>
Or, Get it from RawGit. 
                    <!-- Use this for production -->
                    <script src="https://cdn.rawgit.com/Hkh12/agent/86c1ca8a/src/agent.min.js"></script>

                    <!-- Use this for development -->
                    <script src="https://rawgit.com/Hkh12/agent/master/src/agent.min.js"></script>
How To Use
Start the work like this: 
                    const myagent = new Agent();
Now, You can see all information in myagent variable. 
                    //operating system (object, name and version)
                    myagent.getOs()
                    //browser (object, name and version)
                    myagent.getBrowser()
                    //browser engine (object, name and version)
                    myagent.getBrowsingEngine()
                    //mobile (boolean)
                    myagent.isMobile()
                    //touch device (boolean)
                    myagent.isTouchDevice()
                    //cookies enabled (boolean)
                    myagent.cookieEnabled()
                    //screen info (object, width and height)
                    myagent.screen()
